📖 ツールの使い方

  1. ツールを選択: AI（Gemini等）を選びます。
  2. 被写体を入力: 日本語で入力。画像参照なら空欄でもOK。
  3. 翻訳: 「翻訳」ボタンで英語に変換。
  4. オプション: 画風などを選びます（❓で解説）。
  5. コピーして生成: プロンプトをコピーしたら、スマホのアプリ切り替えで生成AIを開き、画像と一緒に貼り付けて送信！

タイトル

AIごとに最適なプロンプト形式に自動調整します。Midjourneyは単語区切り、Gemini/ChatGPTは自然な文章、Stable Diffusionはタグ形式になります。
画像に描きたいメインの対象です。画像参照メインなら空欄でもAIが画像から推測します。
等身を指定します。
  • ちびキャラ: 2.5頭身の小柄で可愛い体型。
  • 高頭身: 8頭身以上のモデル体型。
基本の表現手法です。
  • 実写写真: カメラ撮影のようなリアル感。
  • アニメ: 2Dセル画風のイラスト。
  • 油絵/水彩画: 伝統的な絵画の質感。
  • 3Dレンダー: 立体的なCG表現。
  • 線画: デッサンやラフスケッチ風。
世界観を付加します。
  • サイバーパンク: ネオン、未来都市。
  • スチームパンク: 歯車、蒸気機関。
  • ダークファンタジー: 重厚な魔法世界。
  • ポップアート: 派手で現代的なアート。
  • ミニマリズム: シンプルな美しさ。
  • レトロ: 懐かしい雰囲気。
光の当たり方です。
  • シネマティック: ドラマチックな明暗。
  • 自然光: 柔らかい陽射し。
  • スタジオ照明: 均一で綺麗な光。
  • ネオン: 雰囲気のある光。
人物の映る範囲です。
  • 全身: 頭から足元まで全体。
  • 上半身: 腰から上。
  • バストアップ: 胸から上（表情強調）。
撮影角度です。
  • 広角: 背景を広く入れた時。
  • 俯瞰: 上から見下ろす視点。
  • あおり: 下から見上げる視点。
高画質・高精細にするための呪文です。
  • 最高品質: 全体的な完成度を底上げします。
  • ArtStation風: プロのような洗練された塗りと構図。
  • UE5風: リアルな光や空気感を強調。
画像の縦横比です。AIツールに合わせて最適なパラメータに変換されます。
  • 1:1: 正方形。SNSのアイコンなどに最適です。
  • 16:9: 横長。風景や映画風に。
  • 9:16: 縦長。スマホ全画面や立ち絵に。
  • 3:2 / 2:3: 標準的な写真サイズ。
背景を元の画像に合わせるか、選んだスタイルに合わせて新しく作るかを選べます。
